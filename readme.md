# Data Science Internship Projects

This repository contains machine learning projects completed during my Data Science Internship.

## Projects Included

1. Iris Flower Classification
- Built a classification model using the Iris dataset.
- Used machine learning algorithms to classify flower species.
- Evaluated model performance using accuracy score.

2. Email Spam Detection
- Developed a text classification model to detect spam emails.
- Applied NLP techniques and machine learning algorithms.
- Evaluated model using accuracy and confusion matrix.

3. Car Price Prediction
- Built a regression model to predict car prices.
- Performed data preprocessing and feature selection.
- Evaluated model performance using R² score.

4. Sales Prediction
- Developed a regression model to predict sales.
- Used real-world dataset.
- Achieved R² score of 0.90+.

## Tools & Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib / Seaborn
- Scikit-learn
- Jupyter Notebook



These projects demonstrate my understanding of machine learning, data preprocessing, model training, and evaluation techniques.
